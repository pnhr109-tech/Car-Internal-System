# 社内システム UI 基本ルールブック

## ― ネイビー基調・確定版（カラーコード付き） ―

---

# 1. カラーパレット定義（固定値）

## 1.1 ベースカラー（ニュートラル）

| 用途        | カラー      | HEX       |
| --------- | -------- | --------- |
| 背景        | ホワイト     | `#FFFFFF` |
| ページ背景（薄）  | ライトグレー   | `#F5F7FA` |
| ボーダー      | グレー      | `#E5E7EB` |
| テキスト（メイン） | ダークグレー   | `#1F2937` |
| テキスト（補助）  | ミディアムグレー | `#6B7280` |

---

## 1.2 アクセント（Primary：ネイビー）

> 低彩度・落ち着き・上長ウケ重視

| 状態     | HEX       | 用途               |
| ------ | --------- | ---------------- |
| 通常     | `#1E3A5F` | Primaryボタン / 選択中 |
| Hover  | `#24476F` | ホバー状態            |
| Active | `#162B45` | 押下時              |
| 枠線用    | `#1E3A5F` | Secondaryボタン枠    |

※ Hoverは「明度を少し上げる」ことで自然な反応感を出す
※ Activeは暗くして押下感を出す

---

## 1.3 ステータスカラー（意味固定）

| 状態   | HEX       | 用途      |
| ---- | --------- | ------- |
| 成功   | `#16A34A` | 承認・完了   |
| 成功背景 | `#DCFCE7` | 成功ラベル背景 |
| 注意   | `#F59E0B` | 期限間近    |
| 注意背景 | `#FEF3C7` | 注意ラベル背景 |
| 失敗   | `#DC2626` | エラー・却下  |
| 失敗背景 | `#FEE2E2` | エラー背景   |
| 情報   | `#0284C7` | 通知      |
| 情報背景 | `#E0F2FE` | 情報表示    |

---

# 2. タイポグラフィ（確定仕様）

| ----- | ---------- |

| 基本文字  | 14px       |

---

# 3. 余白・形状ルール

| 要素       | 値       |
| -------- | ------- |
| ボタン高さ    | 40px    |
| 角丸       | 6px     |
| 入力欄高さ    | 40px    |
| セクション間余白 | 24px以上  |
| 項目間余白    | 12〜16px |

---

# 4. ボタン仕様（完全定義）

## 4.1 Primaryボタン

| 状態       | 背景        | 文字        |
| -------- | --------- | --------- |
| 通常       | `#1E3A5F` | `#FFFFFF` |
| Hover    | `#24476F` | `#FFFFFF` |
| Active   | `#162B45` | `#FFFFFF` |
| Disabled | `#CBD5E1` | `#FFFFFF` |

**ルール**

* 1画面1つまで
* 主操作のみ使用
* 危険操作では使わない

---

## 4.2 Secondaryボタン

| 状態     | 背景        | 枠線        | 文字        |
| ------ | --------- | --------- | --------- |
| 通常     | `#FFFFFF` | `#1E3A5F` | `#1E3A5F` |
| Hover  | `#F0F4F8` | `#1E3A5F` | `#1E3A5F` |
| Active | `#E2E8F0` | `#162B45` | `#162B45` |

用途：

* 戻る
* キャンセル
* 補助操作

---

## 4.3 危険ボタン

| 状態    | 背景        | 文字        |
| ----- | --------- | --------- |
| 通常    | `#DC2626` | `#FFFFFF` |
| Hover | `#B91C1C` | `#FFFFFF` |

# 5. メニュー仕様

### 左メニュー

| ----- | --------- | --------- |
| 通常    | `#FFFFFF` | `#1F2937` |
| Hover | `#F5F7FA` | `#1F2937` |

---

# 6. テーブル仕様

* ヘッダー背景：`#F5F7FA`
* 行Hover：`#F9FAFB`
* ボーダー：`#E5E7EB`
* ステータス表示は背景付きラベルで表現

---

# 7. フォーカス状態（重要）

アクセシビリティ確保のため：

```
outline: 2px solid #1E3A5F;
outline-offset: 2px;
```

---

# 8. やってはいけないこと

❌ ネイビーを装飾として多用
❌ Primaryボタンを複数設置
❌ ステータス色をボタン主色に使用
❌ グラデーション使用
❌ 強い影をつける

---

# 9. トーン最終イメージ

* 白を主役
* ネイビーは締め色
* 情報は整理されている
* 役員層が見ても違和感がない
* 印刷しても違和感がない
