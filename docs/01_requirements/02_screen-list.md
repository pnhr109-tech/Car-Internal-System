# 画面一覧

## メイン画面と派生画面

| メイン画面 | 取引画面（派生画面） | 備考 |
|-----------|---------------------|------|
| トップページ（ダッシュボード） | サイドメニューから各一覧ページに派生 | 一旦査定申込のところだけ実装し、他は工事中 |
| 査定申込一覧 | 査定申込手動入力画面 / 査定申込詳細画面 / 査定申込編集画面 | 一覧画面の「新規追加」ボタンから手動入力画面へ遷移。申込番号クリックで詳細画面へ遷移。詳細画面の編集ボタンで編集画面へ遷移。詳細画面から案件昇格（商談に進む）が可能。一覧では商談昇格済レコードの対応ステータス列に案件ステータス（査定中/成約/没等）を表示。 |
| 案件一覧 | 各商談・契約一気通貫ページ ※そのページ内で商談・契約をタブで確認・変更・登録できるようにすること | 営業は自分の案件すべてを表示（各項目で検索可能）。管理者権限を持つ人はすべて表示（名前や各項目で検索可能） |
| 契約一覧 | 各商談・契約一気通貫ページ | 案件一覧から契約に進んだもののみ表示（DBから取得条件を絞って表示） |

## 実装フェーズ方針

### フェーズ1（現在対象）: アポイント〜契約
- 査定申込一覧（+ 手動入力・詳細変更）
- 案件一覧（商談・契約タブ一気通貫）
- 契約一覧

### フェーズ2以降（段階的に実装）
- 車両引き取り / 出品準備 / 陸送 / オークション / 自社入金 / お客様へ支払い / Web掲載
- ダッシュボードの各指標（現状は一部のみ表示）
