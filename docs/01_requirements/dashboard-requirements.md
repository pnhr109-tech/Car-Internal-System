# トップページ（ダッシュボード）表示項目定義

> 元資料：`トップページ表示項目.xlsx`
> 対象画面：トップ（ダッシュボード）

## 列定義

| 列名 | 説明 |
|---|---|
| 現状のツール | 現在どのツールで管理しているか |
| 項目名 | 表示する情報の名称 |
| 表示対象 | 閲覧権限（営業/CC/管理者/全社） |
| Mock表示 | Mockでの表示番号または画面名 |
| 新システム想定機能 | 新システムでの実装方針 |

---

## 1. 査定申込・新着情報

| 現状のツール | 項目名 | 表示対象 | Mock表示 | 新システム想定機能 |
|---|---|---|---|---|
| ナビクル、マイカー、HP、メール、来店 | 新着情報 | 営業/CC | ① | ナビクルからの新着を自動収集して通知。社内システムから直接電話・予定追加・ステータス変更が可能。申込番号は媒体-内部番号形式（例：N-0001）で表示。 |
| ナビクル、マイカー、HP、メール、来店 | 全査定申込情報 | 営業/CC | 査定申込一覧画面 | 各査定申込サービスの情報を統合保持 |
| 無し | 商談顧客情報 | 営業/CC | 顧客情報画面 | 顧客情報をDBで持ち参照可能にする |
| マイカースカウト／チャット | 新着情報（マイカースカウト） | 管理者 | 査定申込一覧画面 | ナビクル新着と同様にトップページに表示。棲み分け要検討 |
