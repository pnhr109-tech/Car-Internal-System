## 1. 査定申込管理

### 1-1. 申込一覧表示
- 全チャネル（ナビクル / マイカースカウト / カービュー / HP / 来店 / 紹介 / メール / 手動）の申込を一覧表示
- 営業は自分担当のみ表示。管理者は全件表示
- 検索・絞り込み：チャネル種別 / 対応ステータス / 担当者 / 申込日 / 顧客名 / 電話番号

### 1-2. 申込手動入力
- 外部媒体以外（来店・紹介・メール等）の申込を手動で登録するフォーム
- 顧客情報・車両情報・チャネル種別・査定予約日時を入力

### 1-3. 申込詳細・変更
- 申込情報の詳細確認
- ステータス変更（未対応→対応中→商談確定 など）
- 担当者アサイン・変更
- キャンセル理由の記録（没処理）
- 取引・連絡履歴の追記

### 1-4. 商談昇格
- 申込詳細画面から「商談に進む」ボタンで `Assessment` レコードを生成
- 申込の `follow_status` を「商談昇格済」に更新
- 顧客・車両が未紐付けの場合は紐付け画面を経由

### 1-5. Gmail連携（既存機能の維持）
- Gmail Push通知による申込自動取り込み（既存）
- 取り込み済みレコードに `channel_type` を付与するマイグレーション対応

---

## 2. 顧客管理

### 2-1. 顧客登録
- 申込手動入力時 or 申込詳細から顧客を新規登録
- 既存顧客検索（電話番号・氏名）で重複登録防止

### 2-2. 顧客情報編集
- 氏名・電話番号・メール・住所・年齢・職業・性別・家族構成

### 2-3. 口座情報管理
- 顧客に紐付く振込口座の登録・変更
- 複数口座対応、優先口座フラグ

---

## 3. 車両管理

### 3-1. 車両登録
- 申込手動入力時 or 申込詳細から車両を登録
- 既存車両への紐付けも可能

### 3-2. 車両情報編集
- ①アポ時：メーカー / 車種 / 年式 / 走行距離 / グレード / カラー
- ②商談時追加：車台番号 / 修復歴フラグ / 車検有効期限 / ミッション / ナンバー

### 3-3. 車両画像管理
- 画像アップロード（外装 / 内装 / エンジン等、パーツ種別付き）
- 画像一覧・削除

---

## 4. 査定（商談）管理

### 4-1. 査定記録登録
- 商談昇格後に査定内容を入力
- 査定額 / 市場相場価格 / 総合評価 / 査定チェック項目

### 4-2. 査定チェック項目入力
- 傷 / 修復歴 / 内装 / タイヤ / その他の各チェックと詳細説明

### 4-3. 査定ステータス管理
- ステータス遷移：査定中 → 成約 / 不成約 / 管理
- 不成約・管理の場合：キャンセル理由・方針（再アプローチ等）を記録

### 4-4. 査定承認
- マネージャー / 次席による承認操作
- 承認待ち一覧の確認
- 承認・差し戻しアクション

### 4-5. 契約昇格
- 査定「成約」後に「契約へ進む」ボタンで `PurchaseContract` レコードを生成

---

## 5. 買取契約管理

### 5-1. 契約情報登録
- 契約日 / 買取確定価格（税抜・税込） / 支払い予定日 / 出品予定日
- 加修フラグ・所有権解除フラグ・金額訂正フラグ

### 5-2. 契約情報編集
- 上記項目の変更
- 金額訂正の場合は訂正後価格・理由を記録し再承認フローへ

### 5-3. 契約ステータス管理
- 未契約 → 契約済 → 破棄
- 破棄時はキャンセル理由・日時を記録

### 5-4. 契約承認
- 社長 / マネージャー / 次席による承認
- 承認待ち一覧

### 5-5. 所有権解除管理
- 解除フラグONの場合にパターン（A/B）選択
- 各ステータス（残債照会中 / 書類送付済 / 残債振込済 / 書類返却済）の更新

### 5-6. 先払い入金記録
- 前払い入金の登録・社長稟議フラグ管理

---

## 6. 書類管理

### 6-1. 書類一覧（契約紐付き）
- 契約に必要な書類種別の一覧表示
- 各書類のステータス（未作成 / 受領待 / 受領済 / 確認済）確認

### 6-2. 書類ステータス更新
- 各書類の受領・確認操作
- 後日品の送付ステータス管理

### 6-3. 本人確認書類
- 書類種別（運転免許証 / パスポート等）の選択
- 画像ファイルアップロード・確認日時の記録

---

## 7. 取引・連絡履歴

### 7-1. 履歴追記
- 連絡方法（電話 / メール / SMS / 対面）と内容を記録
- 申込詳細・査定詳細・契約詳細の各画面から追記可能

### 7-2. 履歴一覧表示
- 申込に紐付く全履歴を時系列で表示

---

## 8. 承認フロー

### 8-1. 承認待ち一覧（管理者向け）
- 査定承認待ち / 契約承認待ち を一覧表示
- 種別・担当者・申込日でフィルタリング

### 8-2. 承認・差し戻し操作
- 承認 → ステータスを次フェーズへ更新
- 差し戻し → 理由を入力して担当者へ通知

---

## 9. 通知

### 9-1. ステータス変更通知（Googleチャット）
- 査定申込 → 商談昇格 時に自動通知
- 商談 → 契約 時に自動通知

### 9-2. 承認依頼通知
- 査定・契約が承認待ちになった際に承認者へ通知（Googleチャット or システム内通知）

### 9-3. 新着申込通知（既存）
- Gmail Push通知経由の新着申込を通知（既存機能維持）
